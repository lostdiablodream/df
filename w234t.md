## Главное: OpenHands работает через API, а не через claude.ai подписку

Подписки claude.ai (Free, Pro, Max) **отдельны** от API — API всегда тарифицируется по токенам, а не фиксированной платой. OpenHands подключается напрямую через API-ключ, поэтому для максимальной интенсивной работы в OpenHands релевантны **API-тиры**, а не подписки чата.

**Вывод одной строкой:** для крайне интенсивной работы в OpenHands — **API Tier 4 напрямую**, и при достижении его потолка — **Custom Enterprise через sales@anthropic.com**. У Custom-тира нет фиксированных лимитов — они полностью договорные, ограниченные только инфраструктурной ёмкостью, согласованной с Anthropic.
